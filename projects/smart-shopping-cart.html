From Code to Cart

Dataset Creation/Labeling

Using Make Sense to create comprehensive training datasets for accurate object detection and product recognition.

The Physical Build

Arduino integration with servo motors, sensors, and wiring systems seamlessly embedded within the cart structure.

The UI/UX

Clean tablet interface displaying running totals, item details, and payment options for intuitive user interaction.

The Final Prototype

Complete, functional cart with integrated AI detection, real-time billing, and secure payment mechanisms.

Benefits and Evolution

Value Today

The Road Ahead

Tangible Computing in Action

This project bridged the digital and physical worlds. The core challenge was seamlessly integrating the AI's digital detection with the tangible feedback of sensors, and a physical locking mechanism, creating a system that feels intuitive and trustworthy for the user.
